== Description ==

The new version of Wordpress 3.5 to change the media page, removing two fields for defining the location and the sub-field of media.
There is a constant wp-config.php, but it does not include sub-domains.
This plug-in is designed for installations WP virgins.
Once activated, the plugin will add two values ​​in the database you can change it.
Once complete you can disable the plug-in and remove it. It only works activation.

== Installation ==

Enable plug-ins installed on a new WP.
It will redirect you to the options page.
Change the desired values.
Well, it's done!

You can disable the plugin and delete it, it will do nothing more in time.
